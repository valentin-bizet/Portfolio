Industrial Infrastructure Hardening
Automotive Sector • Compliance
TISAX / ISO 27001 Network Segmentation Active Directory
THE CHALLENGE

Migration and security hardening of an industrial infrastructure (270+ nodes) to meet strict TISAX automotive standards. The system required zero downtime during production hours.

ENGINEERING & STRATEGY
  • → Governance: Conducted gap analysis against TISAX controls to identify vulnerabilities in the legacy network.
  • → Implementation: Deployed VLAN segmentation to isolate OT (Operational Tech) from IT networks. Hardened Active Directory GPOs.
  • → Outcome: Successful audit validation with compliant security posture and improved availability.

Secure IoT Telemetry Architecture
Research Project • Network Security
PKI & Certificates TLS 1.3 MQTT
THE CHALLENGE

Securing the transmission of sensitive telemetry data from IoT sensors to a central server over an untrusted public network.

ENGINEERING & STRATEGY
  • → Encryption: Implementation of a full PKI (Public Key Infrastructure) to manage X.509 certificates for mutual authentication (mTLS).
  • → Protocol: Hardening of the MQTT protocol using TLS 1.3 to prevent Man-in-the-Middle attacks.
  • → Outcome: Proof of Concept demonstrating 100% encrypted data flow with authenticated devices only.
